# Hypertension Management in Miami's Street Medicine Initiative

Companion website for a retrospective cohort study of blood pressure outcomes, care engagement, and attrition among unsheltered adults served by Miami Street Medicine (February 2021 – January 2025).

**Live site:** https://street-medicine-hypertension-dannyb685s-projects.vercel.app

## About

The site presents the study in four sections: background, methods (including the screening-to-retention funnel), results (blood pressure outcomes by baseline acuity, intervention intensity, severity transitions, and a durability check), and discussion (the case for street medicine as a reliable bridge to longitudinal care, housing, and social supportive services).

Every statistic shown is drawn from the manuscript. The "follow-up in context" comparison figure draws on external published benchmarks, each cited in the figure caption.

## Technical

`index.html` is the entire site: one self-contained file with all CSS, JavaScript, and SVG icons inline. There is no build step and no package dependencies. The only external request is to Google Fonts.

To view locally, open `index.html` in any browser.

## Deployment

Deployed as a static site on Vercel. No build command, no output directory, framework preset "Other". The repository root is the deploy root.
